📊 Comprehensive Complexity Analysis

🔍 Brute Force Algorithm

Time Complexity: O(b^d)
Space Complexity: O(b^d)
Where: b = branching factor (~5), d = depth (~11)
How it works:
• Explores ALL possible paths using BFS
• Stores ALL visited states in memory
• Guarantees optimal solution
• No pruning of invalid paths early

🎯 Backtracking Algorithm

Time Complexity: O(b^d) worst case, O(d) best case
Space Complexity: O(d)
Where: b = branching factor (~5), d = depth (~11)
How it works:
• Uses DFS with intelligent pruning
• Backtracks when invalid state found
• Uses O(d) space on call stack
• Prunes invalid branches early
Aspect Brute Force (BFS) Backtracking (DFS) Winner
Time Complexity O(5^11) ≈ 48M operations O(5^11) worst, O(11) average 🎯 Backtracking
Space Complexity O(5^11) - stores all states O(11) - only call stack 🎯 Backtracking
Memory Usage Very High (exponential) Low (linear in depth) 🎯 Backtracking
Solution Quality Guaranteed optimal May not be optimal 🔍 Brute Force
Early Termination No - explores all paths Yes - stops at first solution 🎯 Backtracking
Practical Performance Slower for large problems Much faster in practice 🎯 Backtracking

🏆 Why Backtracking is Better Despite Same Big-O

1. Average Case Performance: While both have O(b^d) worst-case complexity, backtracking's average case is much better due to pruning.

2. Space Efficiency: Backtracking uses O(d) space vs O(b^d) for brute force - this is crucial for scalability.

3. Early Termination: Backtracking stops at the first valid solution, while brute force explores everything.

4. Practical Constraints: Real-world problems often hit memory limits with brute force before time limits.

5. Branch Pruning: Invalid states are detected and pruned immediately, reducing the effective branching factor.

struct State {
    int cL, mL, boat, cR, mR; // Cannibals Left, Missionaries Left, Boat Side (0=L, 1=R), Cannibals Right, Missionaries Right
    State* parent;
    std::string action;
    State(int cL, int mL, int boat, int cR, int mR, State* parent = nullptr, const std::string& action = "")
        : cL(cL), mL(mL), boat(boat), cR(cR), mR(mR), parent(parent), action(action) {}

    bool isValid() const {
        if (cL < 0 || mL < 0 || cR < 0 || mR < 0) return false;
        if (cL > 3 || mL > 3 || cR > 3 || mR > 3) return false; // Assuming 3 cannibals and 3 missionaries

        if (mL > 0 && cL > mL) return false;
        if (mR > 0 && cR > mR) return false;
        
        return true;
    }

    bool isGoal() const {
        return cL == 0 && mL == 0 && boat == 1;
    }

    std::string getKey() const {
        return std::to_string(cL) + "," + std::to_string(mL) + "," + std::to_string(boat) + "," + std::to_string(cR) + "," + std::to_string(mR);
    }
};

std::vector getSuccessors(const State& currentState) {
    std::vector successors;
    std::vector> moves = {{1, 0}, {2, 0}, {0, 1}, {0, 2}, {1, 1}}; // {cannibals, missionaries}

    for (auto move : moves) {
        State newState = currentState;
        std::string action;

        if (currentState.boat == 0) { // Boat on left, moving to right
            newState.cL -= move.first;
            newState.mL -= move.second;
            newState.boat = 1;
            newState.cR += move.first;
            newState.mR += move.second;
            action = "Move " + std::to_string(move.first) + " cannibals and " + std::to_string(move.second) + " missionaries from LEFT to RIGHT";
        } else { // Boat on right, moving to left
            newState.cL += move.first;
            newState.mL += move.second;
            newState.boat = 0;
            newState.cR -= move.first;
            newState.mR -= move.second;
            action = "Move " + std::to_string(move.first) + " cannibals and " + std::to_string(move.second) + " missionaries from RIGHT to LEFT";
        }

        newState.parent = new State(currentState); // Create a new parent to avoid dangling pointers
        newState.action = action;
        if (newState.isValid()) {
            successors.push_back(newState);
        } else {
            delete newState.parent; // Clean up invalid state's parent
        }
    }
    return successors;
}
